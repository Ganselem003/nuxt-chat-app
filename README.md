# Nuxt + Go WebSocket Chat App
==============================
- Frontend: **Nuxt 3 + TypeScript + Vue 3** ашигласан. 
- Backend: **Go + Gorilla WebSocket + GORM + SQLite** ашигласан.

# Features
==========
- User authentication: **Register & Login**
- Online users list
- Direct messages (DM) between users
- Real-time WebSocket communication
- Clean UI/UX
- Fully TypeScript-supported frontend

# Run Backend
=============
- go run ./cmd/server

# Run Frontend
==============
- npm run dev
